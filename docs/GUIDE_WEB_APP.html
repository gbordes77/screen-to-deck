Guide Utilisateur — Application Web

Glissez-déposez une capture de deck MTG; l’app extrait les cartes, valide via Scryfall et propose des exports.

Lancement

  1. Installer et valider: 
    bash scripts/bootstrap-local.sh
  2. Clé OpenAI (si mode online): 
    bash scripts/secret-store.sh set sk-...
  3. Démarrer: 
    ./scripts/arrivee.sh
# ou
./scripts/one-button.sh

Accès

  • UI: http://<IP_LAN>:5173
  • API: http://<IP_LAN>:3001 (/health, /api/health, /api/docs, /metrics)

Parcours utilisateur

  1. Page d’accueil → “Convertir une capture”
  2. Déposez votre image (png/jpg/webp) → Envoi à /api/ocr/upload
  3. L’app affiche un identifiant de traitement puis suit /api/ocr/status/:id
  4. À la fin: affichage du deck, corrections/warnings et actions d’export

Exports

  • POST /api/export/:format avec format ∈ {mtga, moxfield, archidekt, tappedout, txt}
  • Prévisualisation: /api/export/preview/:format
  • Téléchargement: /api/export/download/:format

Dépannage

  • Vérifier /health et /api/health
  • OpenAPI: /api/docs doit charger
  • Budget: si limite atteinte, le serveur retourne 429/402 (budgetGuard)
  • Mode offline: sans clé OpenAI, l’API peut basculer EasyOCR/Tesseract
Note: Dans cette version, l’UI inclut des écrans minimaux pour garantir le build. Les écrans complets peuvent être rétablis/ajoutés sans changer le flux.
